## 🔧 技术细节

### 为什么还是超时？

尽管配置了 120 秒超时，bcmss 和 hlbdocs 仍然无法完成克隆。可能原因：

1. **网络链路问题**
   - 中国到 GitHub 连接本身不稳定
   - ISP 可能有限制或丢包

2. **DNS 解析问题**
   - github.com 解析可能不稳定
   - 不同时间段连接质量不同

3. **GitHub API 限制**
   - 短时间内多个克隆请求
   - 可能触发 GitHub 的速率限制

4. **本地网络问题**
   - WiFi 信号不稳定
   - 带宽不足
